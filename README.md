## 报告特点

- **生命周期驱动分析**：先判断行业所处阶段（导入期/成长期/成熟期/衰退期），再动态调整分析重心
- **结构化框架**：涵盖行业概览、护城河评估、竞争格局、PEST 宏观分析、估值与景气度
- **交互式图表**：支持 tooltip、缩放等交互操作
- **数据驱动**：基于公开金融数据与行业研究信息
