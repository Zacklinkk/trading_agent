## 报告特点

- **生命周期驱动分析**：先判断行业所处阶段（导入期/成长期/成熟期/衰退期），再动态调整分析重心
- **结构化框架**：涵盖行业概览、护城河评估、竞争格局、PEST 宏观分析、估值与景气度
- **交互式图表**：基于 Apache ECharts，支持 tooltip、缩放等交互操作
- **数据驱动**：通过 Tushare / AKShare 获取实时金融数据，结合 Web 搜索获取定性信息

## 图表类型

每份报告包含 6 类可视化图表：

- 市场规模趋势（柱状图 + 折线图）
- 护城河评估（雷达图）
- 竞争对手对比（雷达图）
- 营收利润趋势（组合图）
- 业务结构（饼图）
- 毛利率/利润率趋势（折线图）

## 技术栈

- **数据获取**：Tushare Pro API / AKShare
- **可视化**：Apache ECharts 5
- **报告格式**：单文件 HTML（CSS/JS 内联，零外部依赖）
